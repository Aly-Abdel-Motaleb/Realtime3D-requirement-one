This project is a Three.js XR kitchen scene that places a textured robot in a realistic kitchen environment with configurable lighting, materials, and animation. It supports desktop orbit controls as well as WebXR VR and AR modes, including controller-based teleoperation, AR scene placement, and transparent passthrough rendering. The repository includes five exercise pages, a local server, and the assets needed to run and test the scene in a browser or XR emulator.
